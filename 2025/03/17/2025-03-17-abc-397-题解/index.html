C - Variety Split Easy

题目大意

给定一个长度为 NNN 的整数序列：A=(A1,A2,…,AN)A = (A_1, A_2, \dots, A_N)A=(A1​,A2​,…,AN​)。

将 AAA 分成两个非空的连续子数组，求这两个子数组中不同整数的个数之和的最大值。

更正式地，给定一个整数 iii，其中 1≤i≤N−11 \leq i \leq N-11≤i≤N−1，要求求出如下两个值的和的最大值：

  1. 子数组 (A1,A2,…,Ai)(A_1, A_2, \dots, A_i)(A1​,A2​,…,Ai​) 中不同整数的个数。
  2. 子数组 (Ai+1,Ai+2,…,AN)(A_{i+1}, A_{i+2}, \dots, A_N)(Ai+1​,Ai+2​,…,AN​) 中不同整数的个数。

Constraints

  • 2≤N≤3×1052 \leq N \leq 3 \times 10^52≤N≤3×105
  • 1≤Ai≤N1 \leq A_i \leq N1≤Ai​≤N (1≤i≤N1 \leq i \leq N1≤i≤N)
  • All input values are integers.

思路

可以用两个数组 pre 和 bak 分别表示 从 1∼i1 \sim i1∼i 子段不同整数个数之和 和 从 i∼ni \sim ni∼n 子段不同整数个数之和。

通过上面的预处理, 我们便可以快速得知每个切割位置得到两个子段中不同整数个数和, 因此, 遍历每个可切割位点, 更新答案即可。

F - Variety Split Hard

题目大意

给定一个长度为 NNN 的整数序列：A=(A1,A2,…,AN)A = (A_1, A_2, \dots, A_N)A=(A1​,A2​,…,AN​)

将 AAA 分成三个非空的连续子数组，求这三个子数组中不同整数的个数之和的最大值。

更正式地，给定一对整数 (i,j)(i, j)(i,j)，其中 1≤i<j≤N−11 \leq i < j \leq N-11≤i<j≤N−1，要求求出如下三个值的和的最大值：

  1. 子数组 (A1,A2,…,Ai)(A_1, A_2, \dots, A_i)(A1​,A2​,…,Ai​) 中不同整数的个数。
  2. 子数组 (Ai+1,Ai+2,…,Aj)(A_{i+1}, A_{i+2}, \dots, A_j)(Ai+1​,Ai+2​,…,Aj​) 中不同整数的个数。
  3. 子数组 (Aj+1,Aj+2,…,AN)(A_{j+1}, A_{j+2}, \dots, A_N)(Aj+1​,Aj+2​,…,AN​) 中不同整数的个数。

Constraints

  • 3≤N≤3×1053 \leq N \leq 3 \times 10^53≤N≤3×105
  • 1≤Ai≤N1 \leq A_i \leq N1≤Ai​≤N (1≤i≤N1 \leq i \leq N1≤i≤N)
  • All input values are integers.

思路

类似于 C 题的思路, 预处理出 pre (从 1∼i1 \sim i1∼i 子段不同整数个数之和) 和 bak (从 i∼ni \sim ni∼n 子段不同整数个数之和) 数组。

接下来，我们面临的问题是：在 C 题中，只有一个切割点，我们可以在 O(n)\mathcal O(n)O(n) 的时间内求得答案。但是本题中，存在两个位置的切割点，O(N2)\mathcal O(N^2)O(N2) 的时间复杂度，难以快速求解答案，无法满足我们的需求。

我们需要先固定一个切割点(右切割点)：

记 中间子段 不同整数个数的和 为 f(l, r)

因此，题目转化为了求

观察上面的式子，当 jjj 变为 j+1j+1j+1 时，

若 v[j+1]v[j + 1]v[j+1] 存在于中间子段， 则其加入无法对中间子段的不同整数的个数产生贡献。

而当 v[j+1]v[j + 1]v[j+1] 不存在于中间子段时， 则其加入将对中间子段不同整数个数产生贡献。
