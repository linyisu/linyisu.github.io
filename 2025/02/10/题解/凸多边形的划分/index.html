凸多边形的划分

凸多边形的划分 (链2)

题目描述

给定一个具有 NNN 个顶点的凸多边形，将顶点从 111 至 NNN 标号，每个顶点的权值都是一个正整数。将这个凸多边形划分成 N−2N-2N−2 个互不相交的三角形，试求这些三角形顶点的权值乘积和至少为多少。

输入格式

输入第一行为顶点数 NNN

第二行依次为顶点 111 至顶点 NNN 的权值。

输出格式

输出仅一行，为这些三角形顶点的权值乘积和的最小值。

输入样例

输出样例

提示

对于 100%100 \%100% 的数据，有 N≤50N \leq 50N≤50 ，每个点权值小于 10910^9109 。
